## 二、种植条件

**关键技术参数：**


| 指标       | 标准值     |
| ---------- | ---------- |
| 年均气温   | 15-18℃    |
| 年降水量   | 800-1200mm |
| 昼夜温差   | ≥10℃     |
| 土壤硒含量 | ≥0.2mg/kg |

## 四、选购指南

1. **观色**：果面金黄带放射状红晕
2. **触感**：果蒂按压回弹≥0.3mm
3. **闻香**：清新蜜糖香无酒味
4. **验核**：果核无黑斑
